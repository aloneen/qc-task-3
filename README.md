## Задача 1. Параллельная обработка данных

### Темы:
- Thread
- Executor
- Lambda
- Object

### Условие:
Дан список целых чисел.

### Нужно:
- Разбить список на N частей
- Обработать части параллельно через ExecutorService
- Для каждой части посчитать сумму
- Собрать общий результат

### Требования:
- Использовать Callable
- Корректно завершить executor
- Объяснить, почему результат потокобезопасен

## Задача 3. Кеш с аннотацией

### Темы:
- Annotations
- Reflection
- Object

### Условие:
Создать аннотацию @CacheResult.

### Нужно:
- Метод, помеченный аннотацией, кеширует результат
- Ключ строится из аргументов метода
- Повторный вызов возвращает значение из кеша

- Map + Reflection
